Git commands

To clone the repository go to the folder where you want to store your project, and clone the repository: git clone https://github.com/username/username.github.io

To ignore files create .gitignore file. touch .gitignore Then add files that you would like to ignore. .DS_Store .project *.pyc

To add all files we changed to the staging area: git add -A To add a single file to the staging area: git add filename

If you want to maintain a folder and not the files inside it, just put a . gitignore file in the folder with * as the content. This file will make Git ignore all content from the repository.
